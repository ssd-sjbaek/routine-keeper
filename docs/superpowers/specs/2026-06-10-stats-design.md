# 통계 탭 설계 문서

## 목표

루틴 지킴이 앱에 통계 탭을 추가한다. 월별 루틴 달성 현황을 한눈에 볼 수 있고, 연속 달성 스트릭과 루틴별 달성률을 제공한다. 과거 기록 초기화도 가능하다.

## 아키텍처

`index.html` 단일 파일만 수정. 새로운 Firestore 컬렉션 없음 — 기존 `/users/{uid}/routines/{dateKey}` 문서들을 읽어서 클라이언트에서 집계한다.

## 화면 구조

### 탭 구조
메인 카드 상단에 탭 추가:
```
[  오늘  ] [  📊 통계  ]
```
- 탭은 확장 가능하도록 구현 (나중에 가계부 등 추가 예정)
- 기본 선택 탭: 오늘

### 통계 탭 레이아웃 (위→아래)

**1. 월 네비게이션**
```
‹  2026년 6월  ›        🗑 이달 초기화
```
- `‹` / `›` 버튼으로 이전/다음 달 이동
- 현재 달보다 미래 달은 이동 불가 (›  버튼 비활성화)
- `🗑 이달 초기화`: 확인 다이얼로그 후 해당 달 루틴 체크 기록 삭제

**2. 🔥 스트릭 카드** (파스텔 살구 배경 `#fff5ef`)
```
🔥 5        |   12       |   18
일 연속       이번 달 최장   역대 최장
```
- 연속 달성: 오늘 기준 몇 일 연속으로 루틴을 전체 완료했는지
- 이번 달 최장: 선택된 달 기준 가장 긴 연속 완료 구간
- 역대 최장: 전체 기록 중 가장 긴 연속 완료 구간

**3. 📅 월간 달성 달력**
- 해당 달의 날짜를 7열 그리드로 표시 (요일 헤더 포함)
- 파스텔 민트 `#b8e6ca` 동그라미: 그날 루틴 전체 완료
- 회색 `#f5f5f5` 동그라미: 미완료 (기록 있음)
- 빈 칸: 아직 지나지 않은 날 (내일 이후)
- 오늘: 민트 테두리 `#a8d8b8`
- 범례: ● 전체 완료 / ○ 오늘 / ● 미완료

**4. 📊 루틴별 달성률**
- 각 루틴마다 해당 달 기준 완료 % + 가로 바 차트
- 완료 횟수 / 해당 달 지난 날짜 수
- 70% 이상: 파스텔 민트 바 `#b8e6ca`, 텍스트 `#5aaa7a`
- 70% 미만: 파스텔 살구 바 `#ffc4a0`, 텍스트 `#f0956a`

**5. 전체 초기화**
- 섹션 하단에 `전체 기록 초기화` 텍스트 버튼
- 2단계 확인: "정말요? 취소할 수 없어요" → 확인 클릭 시 실행
- 모든 `/users/{uid}/routines/` 문서 일괄 삭제

## 데이터 로직

### 집계에 필요한 데이터
- `/users/{uid}/routines/{dateKey}` — `{ "루틴이름": true/false }` 형태
- `currentRoutines` — 현재 루틴 목록 (이미 메모리에 있음)

### 루틴 전체 완료 판정
`currentRoutines`의 모든 항목이 해당 날짜 문서에서 `true`일 때 완료.

### 달성률 계산
`완료 횟수 / min(오늘 날짜, 해당 달 마지막 날)` × 100

### 스트릭 계산
- 오늘부터 역순으로 날짜를 확인하며 연속 완료 일수를 셈
- 역대 최장은 전체 날짜 범위에서 가장 긴 연속 구간

### 성능
해당 달 데이터만 조회 (월별 문서가 최대 31개). 역대 최장 계산 시 전체 컬렉션 조회 필요.
