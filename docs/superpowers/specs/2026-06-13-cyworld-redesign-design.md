# 루틴 지킴이 — 싸이월드 미니홈피 리디자인 스펙

## 개요

기존 앱의 기능은 그대로 유지하면서 전체 UI를 싸이월드 미니홈피 스타일로 교체한다. 사용자가 설정에서 색상을 직접 커스터마이징할 수 있다.

---

## 1. 전체 디자인 방향

**레퍼런스:** 싸이월드 미니홈피 (2000년대 초 한국 SNS)

### 핵심 시각 요소

| 요소 | 스타일 |
|------|--------|
| 배경 | 파스텔 보케(bokeh) — 부드럽게 흐린 원형 색상 오브 여러 개 |
| 메인 프레임 | 흰색 카드, 둥근 테두리(border-radius: 20px), 하늘색 border |
| 헤더 | 연한 그라디언트 배경, 아바타 이모지, 타이틀, 연속 스트릭 뱃지 |
| 탭바 | 오늘 / 📊 통계 (기존 구조 유지) |
| 섹션 | 연한 배경 카드, 얇은 border |
| 하단 네비 | 홈 / 스케줄 / ⚙️ 설정 (아이콘 + 레이블) |

### 색상 철학
- 단색 기반 (그라디언트 최소화)
- 기본값: 하늘색 계열
- 사용자가 설정에서 자유롭게 변경 가능

## 2. 색상 커스터마이징

### 커스터마이징 가능한 요소 (4개)

| 변수 | 설명 | 기본값 |
|------|------|--------|
| `--color-bg` | 배경 보케 색 | `#b2e0f7` |
| `--color-border` | 메인 프레임 테두리 색 | `#88ccee` |
| `--color-accent` | 포인트/버튼/탭 활성 색 | `#4488bb` |
| `--color-card` | 카드 배경 색 | `#ffffff` |

### UI
- 위치: 설정 화면 내 "🎨 색상 꾸미기" 섹션
- 각 항목마다: 프리셋 스와치 5개 + 헥스 코드 직접 입력 필드
- 저장 버튼 → localStorage에 저장 → CSS 변수 즉시 적용
- 앱 로드 시 localStorage에서 불러와 CSS 변수 덮어쓰기

### 프리셋 스와치 (각 항목별 5색)

**배경 보케:**
`#b2e0f7`(하늘) · `#ffd0e8`(핑크) · `#c8f0d8`(민트) · `#f0d8f8`(라벤더) · `#ffeec8`(피치)

**테두리:**
`#88ccee`(하늘) · `#ffaad4`(핑크) · `#88ccb8`(민트) · `#cc99ee`(라벤더) · `#ffbb88`(피치)

**포인트:**
`#4488bb`(블루) · `#cc4477`(핑크) · `#339977`(민트) · `#8855cc`(라벤더) · `#cc7733`(피치)

**카드 배경:**
`#ffffff`(흰색) · `#f8fdff`(아이스) · `#fff8fc`(로즈화이트) · `#f8fff8`(민트화이트) · `#fffaf4`(크림)

## 3. 레이아웃 구조

```
[보케 배경 — 전체 화면]
  ↳ [메인 프레임 카드]
      ↳ [헤더: 아바타 + 타이틀 + 스트릭]
      ↳ [탭바: 오늘 | 📊 통계]
      ↳ [날짜 힌트]
      ↳ [panel-today]
          ↳ [루틴 섹션]
          ↳ [할 일 섹션]
      ↳ [panel-stats] (기존 통계 패널)
      ↳ [하단 네비: 홈 | 📅 스케줄 | ⚙️ 설정]
```

### 변경사항 (기존 vs 신규)

| 기존 | 신규 |
|------|------|
| 단색 흰 배경 | 파스텔 보케 배경 |
| 기본 카드 스타일 | 싸이월드 프레임 스타일 |
| 상단 시프트/날짜 행 | 헤더 아바타 + 스트릭 뱃지 통합 |
| 하단 네비 없음 | 아이콘 + 텍스트 하단 네비 |
| 색상 고정 | CSS 변수 + localStorage 커스터마이징 |

## 4. CSS 아키텍처

- CSS 커스텀 프로퍼티(변수)로 색상 관리
- `<style>` 태그 상단에 `:root { --color-bg: ...; }` 정의
- 앱 로드 시 `applyTheme()` 함수가 localStorage 값으로 덮어씀
- 색상이 파생되는 규칙 (accent 색 기준):
  - 버튼 배경: `--color-accent`
  - 탭 활성: `--color-accent`
  - 섹션 타이틀: `--color-accent`
  - 테두리 연한 버전: `--color-border`에서 직접 사용

---

## 5. 설정 화면

기존 설정 화면(로그인/로그아웃 등)에 "🎨 색상 꾸미기" 섹션 추가.

**섹션 내 구성:**
1. 색상 항목 4개 (각각 스와치 5개 + 헥스 입력)
2. 미리보기 바 (선택한 색상 즉시 반영)
3. 저장하기 버튼

**저장 로직:**
```javascript
localStorage.setItem('theme', JSON.stringify({
  bg: '#b2e0f7',
  border: '#88ccee',
  accent: '#4488bb',
  card: '#ffffff'
}));
```
